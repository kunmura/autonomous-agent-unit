## Approval Gate
When a phase is complete and transitioning to the next:
- Write "approval pending" in status.md
- Do NOT automatically create tasks for the next phase
- Create an approval request PPT and upload to Slack (see below)
- Wait for producer approval

## Approval Request PPT (required when approval gate triggers — director creates it directly)
**IMPORTANT: Do NOT delegate this to a team member. The director must create the PPT directly using Bash with python-pptx.**
This is the sole exception to the "director does not do hands-on work" rule. Delegating PPT creation would create an extra phase and delay approval.

Steps:
1. First write "approval pending" in status.md (activate the gate before creating PPT)
2. Create `team/director/output/approval_request_phaseN.pptx` using the following Python code (N = phase number):

```python
from pptx import Presentation
from pptx.util import Inches, Pt
prs = Presentation()
# Slide 1: Title "Phase N Complete — Approval Request" + project name + date
# Slide 2: What was done in this phase (task list with assignees)
# Slide 3: Key deliverables (file names, summaries, quality assessment)
# Slide 4: Highlights (most important findings/conclusions as bullet points)
# Slide 5: Proposed next phase plan (what and why)
# Slide 6: Approval question (clearly ask "May we proceed to the next phase?")
prs.save('team/director/output/approval_request_phaseN.pptx')
```

3. Upload the .pptx file via Slack files API (v2):
```bash
FILE_PATH="team/director/output/approval_request_phaseN.pptx"
FILE_SIZE=$(stat -f%z "$FILE_PATH" 2>/dev/null || stat -c%s "$FILE_PATH")
RESP=$(curl -s -X POST 'https://slack.com/api/files.getUploadURLExternal' \
  -H "Authorization: Bearer $SLACK_TOKEN" \
  -H 'Content-Type: application/x-www-form-urlencoded' \
  -d "filename=$(basename $FILE_PATH)&length=$FILE_SIZE")
UPLOAD_URL=$(echo "$RESP" | python3 -c "import sys,json; print(json.load(sys.stdin)['upload_url'])")
FILE_ID=$(echo "$RESP" | python3 -c "import sys,json; print(json.load(sys.stdin)['file_id'])")
curl -s -X POST "$UPLOAD_URL" -F "file=@$FILE_PATH"
curl -s -X POST 'https://slack.com/api/files.completeUploadExternal' \
  -H "Authorization: Bearer $SLACK_TOKEN" \
  -H 'Content-Type: application/json' \
  -d "{\"files\":[{\"id\":\"$FILE_ID\",\"title\":\"Approval document\"}],\"channel_id\":\"$SLACK_CHANNEL\",\"initial_comment\":\"Phase N complete. Requesting approval to proceed.\"}"
```
